### Step Installation
1. Download, extract zip.

2. In Terminal project, run composer in terminal.

```bash
composer install
```

3. Setting database at file .env.

4. Run php artisan

```bash
php artisan migrate
```
5. Generate key app

```bash
php artisan key:generate
```

6. Running application

```bash
php artisan serve
```

7. Open new Terminal, Run:

```bash
php artisan reverb:start
```
